# CI4 SEO + AI Library (Drop-in) + FEEDS

CodeIgniter 4 projeleri için arama motorları (Google/Bing/Yandex), LLM/AI crawler’lar
(GPTBot, Perplexity, Google-Extended) ve çoklu **Sitemap** + **Feeds** (RSS/Atom/JSON) desteği.

## İçerik
- `app/` : Drop-in kütüphane (Config, Services, Filters, Helpers, Libraries, Controllers)
- `examples/` : Çoklu senaryo örnekleri
- `USAGE.md` : Ayrıntılı kurulum ve kullanım talimatları
